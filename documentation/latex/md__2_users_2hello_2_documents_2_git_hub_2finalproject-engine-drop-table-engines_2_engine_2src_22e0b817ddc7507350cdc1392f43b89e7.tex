The {\ttfamily \doxylink{class_napi_1_1_escapable_handle_scope}{Napi\+::\+Escapable\+Handle\+Scope}} class is used to manage the lifetime of object handles which are created through the use of node-\/addon-\/api. These handles keep an object alive in the heap in order to ensure that the objects are not collected by the garbage collector while native code is using them. A handle may be created when any new node-\/addon-\/api Value or one of its subclasses is created or returned.

The {\ttfamily \doxylink{class_napi_1_1_escapable_handle_scope}{Napi\+::\+Escapable\+Handle\+Scope}} is a special type of {\ttfamily \doxylink{class_napi_1_1_handle_scope}{Napi\+::\+Handle\+Scope}} which allows a single handle to be "{}promoted"{} to an outer scope.
