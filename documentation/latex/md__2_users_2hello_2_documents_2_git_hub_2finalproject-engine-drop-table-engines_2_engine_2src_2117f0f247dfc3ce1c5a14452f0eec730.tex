The Handle\+Scope class is used to manage the lifetime of object handles which are created through the use of node-\/addon-\/api. These handles keep an object alive in the heap in order to ensure that the objects are not collected while native code is using them. A handle may be created when any new node-\/addon-\/api Value or one of its subclasses is created or returned.
