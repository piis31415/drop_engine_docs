The function {\ttfamily Adjust\+External\+Memory} adjusts the amount of registered external memory used to give the Java\+Script engine an indication of the amount of externally allocated memory that is kept alive by Java\+Script objects. The Java\+Script engine uses this to decide when to perform global garbage collections. Registering externally allocated memory will trigger global garbage collections more often than it would otherwise in an attempt to garbage collect the Java\+Script objects that keep the externally allocated memory alive.
