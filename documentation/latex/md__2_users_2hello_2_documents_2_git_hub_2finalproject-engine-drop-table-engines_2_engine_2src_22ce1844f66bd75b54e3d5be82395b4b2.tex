A function that returns the corresponding Chromium version for a given Electron function. Returns a string.

If you provide it with a major Electron version, it will return a major Chromium version\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ chromeVersion\ =\ e2c.electronToChromium('1.4');}
\DoxyCodeLine{//\ chromeVersion\ is\ "{}53"{}}

\end{DoxyCode}


If you provide it with a full Electron version, it will return the full Chromium version.


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ chromeVersion\ =\ e2c.electronToChromium('1.4.11');}
\DoxyCodeLine{//\ chromeVersion\ is\ "{}53.0.2785.143"{}}

\end{DoxyCode}


If a query does not match a Chromium version, it will return {\ttfamily undefined}.


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ chromeVersion\ =\ e2c.electronToChromium('9000');}
\DoxyCodeLine{//\ chromeVersion\ is\ undefined}
