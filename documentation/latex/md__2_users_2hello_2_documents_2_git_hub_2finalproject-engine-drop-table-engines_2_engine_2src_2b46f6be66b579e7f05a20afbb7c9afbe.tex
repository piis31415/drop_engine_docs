A Type\+Script/\+Java\+Script library and command-\/line tool for attaching license agreements to mac\+OS {\ttfamily .dmg} files, which the user must accept before mounting the disk image. Doing this correctly is surprisingly complicated, so this package is here to automate it.

{\bfseries{Warning\+:}} Attaching license agreements to disk images may become impossible in future versions of mac\+OS. It is done using the {\ttfamily hdiutil udifrez} command, which is deprecated as of mac\+OS 12 with no apparent replacement. See \href{https://github.com/argv-minus-one/dmg-license/issues/11}{\texttt{ issue \#11}} for details.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3881}{}\doxysection{\texorpdfstring{Contents}{Contents}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3881}

Although this package is mainly intended for use by {\ttfamily .dmg}-\/generating tools like \href{https://github.com/LinusU/node-appdmg}{\texttt{ node-\/appdmg}}, it can also be used by itself from the command line, to attach a license agreement to an existing {\ttfamily .dmg} file.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3884}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3884}
You can install this package and use the command-\/line tool by running {\ttfamily npm install -\/-\/global dmg-\/license}. This will add a {\ttfamily dmg-\/license} command to your system.

Alternatively, you can run it without installing, using {\ttfamily npx dmg-\/license}. Note that running it this way is relatively slow.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3885}{}\doxysubsection{\texorpdfstring{Command Syntax}{Command Syntax}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2b46f6be66b579e7f05a20afbb7c9afbe_autotoc_md3885}
