(npm 6 or older only!)}}

\href{https://github.com/nodejs/node-gyp/issues?q=label\%3A}{\texttt{ Many issues}} are opened by users who are not running a \href{https://github.com/nodejs/node-gyp/releases}{\texttt{ current version of node-\/gyp}}.

{\ttfamily npm} bundles its own, internal, copy of {\ttfamily node-\/gyp}. This internal copy is independent of any globally installed copy of node-\/gyp that may have been installed via {\ttfamily npm install -\/g node-\/gyp}.

This means that while {\ttfamily node-\/gyp} doesn\textquotesingle{}t get installed into your {\ttfamily \$PATH} by default, npm still keeps its own copy to invoke when you attempt to {\ttfamily npm install} a native add-\/on.

Sometimes, you may need to update npm\textquotesingle{}s internal node-\/gyp to a newer version than what is installed. A simple {\ttfamily npm install -\/g node-\/gyp} {\itshape won\textquotesingle{}t} do the trick since npm will still continue to use its internal copy over the global one.

So instead\+:\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8213}{}\doxysection{\texorpdfstring{Version of npm}{Version of npm}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8213}
We need to start by knowing your version of {\ttfamily npm}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ -\/-\/version}

\end{DoxyCode}
\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8214}{}\doxysection{\texorpdfstring{Linux, mac\+OS, Solaris, etc.}{Linux, mac\+OS, Solaris, etc.}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8214}
Unix is easy. Just run the following command.

If your npm is version \+\_\+\+\_\+\+\_\+7 or 8\+\_\+\+\_\+\+\_\+, do\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ npm\ explore\ npm/node\_modules/@npmcli/run-\/script\ -\/g\ -\/-\/\ npm\_config\_global=false\ npm\ install\ node-\/gyp@latest}

\end{DoxyCode}


Else if your npm is version \+\_\+\+\_\+\+\_\+less than 7\+\_\+\+\_\+\+\_\+, do\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ npm\ explore\ npm/node\_modules/npm-\/lifecycle\ -\/g\ -\/-\/\ npm\ install\ node-\/gyp@latest}

\end{DoxyCode}


If the command fails with a permissions error, please try {\ttfamily sudo} and then the command.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8215}{}\doxysection{\texorpdfstring{Windows}{Windows}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2293b1e162a2446c3e18a466b5be067cf_autotoc_md8215}
Windows is a bit trickier, since {\ttfamily npm} might be installed to the "{}\+Program Files"{} directory, which needs admin privileges in order to modify on current Windows. Therefore, run the following commands {\bfseries{inside a {\ttfamily cmd.\+exe} started with "{}\+Run as Administrator"{}}}\+:

First we need to find the location of {\ttfamily node}. If you don\textquotesingle{}t already know the location that {\ttfamily node.\+exe} got installed to, then run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ where\ node}

\end{DoxyCode}


Now {\ttfamily cd} to the directory that {\ttfamily node.\+exe} is contained in e.\+g.\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ cd\ "{}C:\(\backslash\)Program\ Files\(\backslash\)nodejs"{}}

\end{DoxyCode}


If your npm version is \+\_\+\+\_\+\+\_\+7 or 8\+\_\+\+\_\+\+\_\+, do\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ node\_modules\(\backslash\)npm\(\backslash\)node\_modules\(\backslash\)@npmcli\(\backslash\)run-\/script}

\end{DoxyCode}


Else if your npm version is \+\_\+\+\_\+\+\_\+less than 7\+\_\+\+\_\+\+\_\+, do\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ node\_modules\(\backslash\)npm\(\backslash\)node\_modules\(\backslash\)npm-\/lifecycle}

\end{DoxyCode}


Finish by running\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ npm\ install\ node-\/gyp@latest}

\end{DoxyCode}
