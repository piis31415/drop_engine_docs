\begin{quote}
Strip comments from JSON. Lets you use comments in your JSON files! \end{quote}
This is now possible\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ \ \ //\ Rainbows}
\DoxyCodeLine{\ \ \ \ "{}unicorn"{}:\ /*\ ❤\ */\ "{}cake"{}}
\DoxyCodeLine{\}}

\end{DoxyCode}


It will replace single-\/line comments {\ttfamily //} and multi-\/line comments {\ttfamily /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}/} with whitespace. This allows JSON error positions to remain as close as possible to the original source.
