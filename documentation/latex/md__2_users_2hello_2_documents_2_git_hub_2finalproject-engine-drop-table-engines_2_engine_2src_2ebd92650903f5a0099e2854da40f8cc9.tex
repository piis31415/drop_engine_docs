{\ttfamily ansi-\/styles} allows converting between various color formats and ANSI escapes, with support for 16, 256 and \href{https://gist.github.com/XVilka/8346728}{\texttt{ 16 million colors}}.

The following color spaces are supported\+:


\begin{DoxyItemize}
\item {\ttfamily rgb}
\item {\ttfamily hex}
\item {\ttfamily ansi256}
\item {\ttfamily ansi}
\end{DoxyItemize}

To use these, call the associated conversion function with the intended output, for example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ styles\ from\ 'ansi-\/styles';}
\DoxyCodeLine{}
\DoxyCodeLine{styles.color.ansi(styles.rgbToAnsi(100,\ 200,\ 15));\ //\ RGB\ to\ 16\ color\ ansi\ foreground\ code}
\DoxyCodeLine{styles.bgColor.ansi(styles.hexToAnsi('\#C0FFEE'));\ //\ HEX\ to\ 16\ color\ ansi\ foreground\ code}
\DoxyCodeLine{}
\DoxyCodeLine{styles.color.ansi256(styles.rgbToAnsi256(100,\ 200,\ 15));\ //\ RGB\ to\ 256\ color\ ansi\ foreground\ code}
\DoxyCodeLine{styles.bgColor.ansi256(styles.hexToAnsi256('\#C0FFEE'));\ //\ HEX\ to\ 256\ color\ ansi\ foreground\ code}
\DoxyCodeLine{}
\DoxyCodeLine{styles.color.ansi16m(100,\ 200,\ 15);\ //\ RGB\ to\ 16\ million\ color\ foreground\ code}
\DoxyCodeLine{styles.bgColor.ansi16m(...styles.hexToRgb('\#C0FFEE'));\ //\ Hex\ (RGB)\ to\ 16\ million\ color\ foreground\ code}

\end{DoxyCode}
