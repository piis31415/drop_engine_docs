{\bfseries{\href{https://www.npmjs.com/package/generator-napi-module}{\texttt{ generator-\/napi-\/module}}}} is a module to quickly generate a skeleton module using {\bfseries{N-\/\+API}}, the new API for Native addons. This module automatically sets up your {\bfseries{gyp file}} to use {\bfseries{node-\/addon-\/api}}, the C++ wrappers for N-\/\+API and generates a wrapper JS module.
