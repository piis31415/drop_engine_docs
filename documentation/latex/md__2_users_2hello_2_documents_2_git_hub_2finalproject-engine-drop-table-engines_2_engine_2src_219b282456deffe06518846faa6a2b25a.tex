The {\ttfamily \doxylink{class_napi_1_1_external}{Napi\+::\+External}} template class implements the ability to create a {\ttfamily \doxylink{class_napi_1_1_value}{Napi\+::\+Value}} object with arbitrary C++ data. It is the user\textquotesingle{}s responsibility to manage the memory for the arbitrary C++ data.

{\ttfamily \doxylink{class_napi_1_1_external}{Napi\+::\+External}} objects can be created with an optional Finalizer function and optional Hint value. The Finalizer function, if specified, is called when your {\ttfamily \doxylink{class_napi_1_1_external}{Napi\+::\+External}} object is released by Node\textquotesingle{}s garbage collector. It gives your code the opportunity to free any dynamically created data.
