Only collaborators in npm for {\bfseries{node-\/addon-\/api}} can create releases. If you want to be able to do releases ask one of the existing collaborators to add you. If necessary you can ask the build Working Group who manages the Node.\+js npm user to add you if there are no other active collaborators.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26410ed06c126d5f5da9383bb59ada580_autotoc_md7755}{}\doxysection{\texorpdfstring{Prerequisites}{Prerequisites}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26410ed06c126d5f5da9383bb59ada580_autotoc_md7755}
Before to start creating a new release check if you have installed the following tools\+:


\begin{DoxyItemize}
\item \href{https://www.npmjs.com/package/changelog-maker}{\texttt{ Changelog maker}}
\end{DoxyItemize}

If not please follow the instruction reported in the tool\textquotesingle{}s documentation to install it.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26410ed06c126d5f5da9383bb59ada580_autotoc_md7756}{}\doxysection{\texorpdfstring{Publish new release}{Publish new release}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26410ed06c126d5f5da9383bb59ada580_autotoc_md7756}
These are the steps to follow to create a new release\+:


\begin{DoxyItemize}
\item Open an issue in the {\bfseries{node-\/addon-\/api}} repo documenting the intent to create a new release. Give people some time to comment or suggest PRs that should land first.
\item Validate all tests pass by running npm test on master.
\item Update the version in {\bfseries{package.\+json}} appropriately.
\item Update the \href{https://github.com/nodejs/node-addon-api/blob/master/README.md}{\texttt{ README.\+md}} to show the new version as the latest.
\item Generate the changelog for the new version using {\bfseries{changelog maker}} tool. From the route folder of the repo launch the following command\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{>\ changelog-\/maker}

\end{DoxyCode}
 ~\newline

\item Use the output generated by {\bfseries{changelog maker}} to pdate the \href{https://github.com/nodejs/node-addon-api/blob/master/CHANGELOG.md}{\texttt{ CHANGELOG.\+md}} following the style used in publishing the previous release.
\item Add any new contributors to the "{}contributors"{} section in the package.\+json
\item Validate all tests pass by running npm test on master.
\item Use {\bfseries{\href{https://ci.nodejs.org/view/x\%20-\%20Abi\%20stable\%20module\%20API/job/node-test-node-addon-api/}{\texttt{ CI}}}} to validate tests pass for latest 11, 10, 8, 6 releases (note there are still some issues on Smart\+OS and Windows in the testing).
\item Do a clean checkout of node-\/addon-\/api.
\item Login and then run {\ttfamily npm publish}.
\item Create a release in Github (look at existing releases for an example).
\item Validate that you can run {\ttfamily npm install node-\/addon-\/api} successfully and that the correct version is installed.
\item Comment on the issue opened in the first step that the release has been created and close the issue.
\item Tweet that the release has been created. 
\end{DoxyItemize}
