The Electron framework lets you write cross-\/platform desktop applications using Java\+Script, HTML and CSS. It is based on \href{https://nodejs.org/}{\texttt{ Node.\+js}} and \href{https://www.chromium.org}{\texttt{ Chromium}} and is used by the \href{https://github.com/Microsoft/vscode/}{\texttt{ Visual Studio Code}} and many other \href{https://electronjs.org/apps}{\texttt{ apps}}.

Follow \href{https://twitter.com/electronjs}{\texttt{ @electronjs}} on Twitter for important announcements.

Each Electron release provides binaries for mac\+OS, Windows, and Linux.


\begin{DoxyItemize}
\item mac\+OS (Big Sur and up)\+: Electron provides 64-\/bit Intel and Apple Silicon / ARM binaries for mac\+OS.
\item Windows (Windows 10 and up)\+: Electron provides {\ttfamily ia32} ({\ttfamily x86}), {\ttfamily x64} ({\ttfamily amd64}), and {\ttfamily arm64} binaries for Windows. Windows on ARM support was added in Electron 5.\+0.\+8. Support for Windows 7, 8 and 8.\+1 was \href{https://www.electronjs.org/blog/windows-7-to-8-1-deprecation-notice}{\texttt{ removed in Electron 23, in line with Chromium\textquotesingle{}s Windows deprecation policy}}.
\item Linux\+: The prebuilt binaries of Electron are built on Ubuntu 20.\+04. They have also been verified to work on\+:
\begin{DoxyItemize}
\item Ubuntu 18.\+04 and newer
\item Fedora 32 and newer
\item Debian 10 and newer
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_25d5d5c380717f0ccbf44833583c21387_autotoc_md4549}{}\doxysubsection{\texorpdfstring{Quick start \& Electron Fiddle}{Quick start \& Electron Fiddle}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_25d5d5c380717f0ccbf44833583c21387_autotoc_md4549}

\DoxyCodeLine{const\ proc\ =\ require('node:child\_process')}

\DoxyCodeLine{//\ will\ print\ something\ similar\ to\ /Users/maf/.../Electron}
\DoxyCodeLine{console.log(electron)}
\DoxyCodeLine{}
\DoxyCodeLine{//\ spawn\ Electron}
\DoxyCodeLine{const\ child\ =\ proc.spawn(electron)}
