The new release of {\ttfamily xmlbuilder} is available at \href{https://github.com/oozcitak/xmlbuilder2}{\texttt{ {\ttfamily xmlbuilder2}}}! {\ttfamily xmlbuilder2} has been redesigned from the ground up to be fully conforming to the \href{https://dom.spec.whatwg.org}{\texttt{ modern DOM specification}}. It supports XML namespaces, provides built-\/in converters for multiple formats, collection functions, and more. Please see \href{https://oozcitak.github.io/xmlbuilder2/upgrading-from-xmlbuilder.html}{\texttt{ upgrading from xmlbuilder}} in the wiki.
