Well, a Universal app isn\textquotesingle{}t anything magical. It is literally the x64 app and the arm64 app glued together into a single application. It\textquotesingle{}s twice as big because it contains two apps in one.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_24bee799b98e95b5ae172fbc051145f19_autotoc_md779}{}\doxysubsubsection{\texorpdfstring{What about native modules?}{What about native modules?}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_24bee799b98e95b5ae172fbc051145f19_autotoc_md779}
The way {\ttfamily @electron/universal} works today means you don\textquotesingle{}t need to worry about things like building universal versions of your native modules.
