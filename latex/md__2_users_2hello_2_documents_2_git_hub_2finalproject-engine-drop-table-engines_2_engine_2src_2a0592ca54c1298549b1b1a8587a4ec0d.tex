Type\+: {\ttfamily Function}

Accepts a function that is called when the promise is canceled.

You\textquotesingle{}re not required to call this function. You can call this function multiple times to add multiple cancel handlers.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8591}{}\doxysubsection{\texorpdfstring{PCancelable\#cancel(reason?)}{PCancelable\#cancel(reason?)}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8591}
Type\+: {\ttfamily Function}

Cancel the promise and optionally provide a reason.

The cancellation is synchronous. Calling it after the promise has settled or multiple times does nothing.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8592}{}\doxysubsection{\texorpdfstring{PCancelable\#is\+Canceled}{PCancelable\#is\+Canceled}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8592}

Whether the promise is canceled.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8593}{}\doxysubsection{\texorpdfstring{PCancelable.\+Cancel\+Error}{PCancelable.\+Cancel\+Error}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2a0592ca54c1298549b1b1a8587a4ec0d_autotoc_md8593}

The function you specify will have {\ttfamily on\+Cancel} appended to its parameters.
