{\ttfamily exhaustive-\/deps} can be configured to validate dependencies of custom Hooks with the {\ttfamily additional\+Hooks} option. This option accepts a regex to match the names of custom Hooks that have dependencies.

\DoxyCodeLine{\}}

\end{DoxyCode}


We suggest to use this option {\bfseries{very sparingly, if at all}}.
