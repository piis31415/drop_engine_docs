First you need to add a reference to boolean in your application\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ \{\ boolean,\ isBooleanable\ \}\ =\ require('boolean');}

\end{DoxyCode}


If you use Type\+Script, use the following code instead\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ \{\ boolean,\ isBooleanable\ \}\ from\ 'boolean';}

\end{DoxyCode}


To verify a value for its boolean value, call the {\ttfamily boolean} function and provide the value in question as parameter\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(boolean('true'));\ //\ =>\ true}

\end{DoxyCode}


The {\ttfamily boolean} function considers the following values to be equivalent to {\ttfamily true}\+:


\begin{DoxyItemize}
\item {\ttfamily true} (boolean)
\item `\textquotesingle{}true'{\ttfamily (string) -\/}\textquotesingle{}TRUE\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}t\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}T\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}yes\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}YES\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}y\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}Y\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}on\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}ON\textquotesingle{}{\ttfamily (string) -\/}\textquotesingle{}1\textquotesingle{}{\ttfamily (string) -\/}1\`{} (number)
\end{DoxyItemize}

In addition to the primitive types mentioned above, boolean also supports their object wrappers {\ttfamily Boolean}, {\ttfamily String}, and {\ttfamily Number}.

{\itshape Please note that if you provide a {\ttfamily string} or a {\ttfamily String} object, it will be trimmed.}

All other values, including {\ttfamily undefined} and {\ttfamily null} are considered to be {\ttfamily false}.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2831c7df6af7fdb0ef452200fd131992e_autotoc_md2639}{}\doxysubsection{\texorpdfstring{Figuring out whether a value can be considered to be boolean}{Figuring out whether a value can be considered to be boolean}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2831c7df6af7fdb0ef452200fd131992e_autotoc_md2639}
From time to time, you may not want to directly convert a value to its boolean equivalent, but explicitly check whether it looks like a boolean. E.\+g., although `boolean(\textquotesingle{}F'){\ttfamily returns}false{\ttfamily , the string}F{\ttfamily at least looks like a boolean, in contrast to something such as}123{\ttfamily (for which}boolean(123){\ttfamily would also return}false\`{}).

To figure out whether a value can be considered to be a boolean, use the {\ttfamily is\+Booleanable} function\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(isBooleanable('true'));\ //\ =>\ true}

\end{DoxyCode}
