Check if the path is executable. If no callback provided, and a global {\ttfamily Promise} object is available, then a Promise will be returned.

Will raise whatever errors may be raised by {\ttfamily fs.\+stat}, unless {\ttfamily options.\+ignore\+Errors} is set to true.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2e928a2a027f992a2396b8671c2ffb37a_autotoc_md6103}{}\doxysubsection{\texorpdfstring{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2e928a2a027f992a2396b8671c2ffb37a_autotoc_md6103}
Same as {\ttfamily isexe} but returns the value and throws any errors raised.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2e928a2a027f992a2396b8671c2ffb37a_autotoc_md6104}{}\doxysubsection{\texorpdfstring{Options}{Options}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_2e928a2a027f992a2396b8671c2ffb37a_autotoc_md6104}

\item {\ttfamily ignore\+Errors} Treat all errors as "{}no, this is not executable"{}, but don\textquotesingle{}t raise them.

\item {\ttfamily gid} Number to use as the group id
