Recursive version of fs.\+readdir wih stream API and glob filtering. Uses the {\ttfamily minimatch} library to do its matching.

Requires Node.\+js 10.\+0 or later.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9063}{}\doxysection{\texorpdfstring{Performances}{Performances}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9063}
Compared to {\ttfamily glob}, {\ttfamily readdir-\/glob} is memory efficient\+: no matter the file system size, or the number of returned files, the memory usage is constant.

The CPU cost is proportional to the number of files in {\ttfamily root} folder, minus the number files in {\ttfamily options.\+skip} folders.

{\bfseries{Advice}}\+: For better performances use {\ttfamily options.\+skip} to restrict the search as much as possible.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9064}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9064}

\begin{DoxyItemize}
\item {\ttfamily pattern}\+: Glob pattern or Array of Glob patterns to match the found files with. A file has to match at least one of the provided patterns to be returned.
\item {\ttfamily ignore}\+: Glob pattern or Array of Glob patterns to exclude matches. If a file or a folder matches at least one of the provided patterns, it\textquotesingle{}s not returned. It doesn\textquotesingle{}t prevent files from folder content to be returned. Note\+: {\ttfamily ignore} patterns are {\itshape always} in {\ttfamily dot\+:true} mode.
\item {\ttfamily skip}\+: Glob pattern or Array of Glob patterns to exclude folders. If a folder matches one of the provided patterns, it\textquotesingle{}s not returned, and it\textquotesingle{}s not explored\+: this prevents any of its children to be returned. Note\+: {\ttfamily skip} patterns are {\itshape always} in {\ttfamily dot\+:true} mode.
\item {\ttfamily mark}\+: Add a {\ttfamily /} character to directory matches.
\item {\ttfamily stat}\+: Set to true to stat {\itshape all} results. This reduces performance.
\item {\ttfamily silent}\+: When an unusual error is encountered when attempting to read a directory, a warning will be printed to stderr. Set the {\ttfamily silent} option to true to suppress these warnings.
\item {\ttfamily nodir}\+: Do not match directories, only files.
\item {\ttfamily follow}\+: Follow symlinked directories. Note that requires to stat {\itshape all} results, and so reduces performance.
\end{DoxyItemize}

The following options apply only if {\ttfamily pattern} option is set, and are forwarded to {\ttfamily minimatch}\+:
\begin{DoxyItemize}
\item {\ttfamily dot}\+: Allow {\ttfamily pattern} to match filenames starting with a period, even if the pattern does not explicitly have a period in that spot.
\item {\ttfamily noglobstar}\+: Disable {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}} matching against multiple folder names.
\item {\ttfamily nocase}\+: Perform a case-\/insensitive match. Note\+: on case-\/insensitive filesystems, non-\/magic patterns will match by default, since {\ttfamily stat} and {\ttfamily readdir} will not raise errors.
\item {\ttfamily match\+Base}\+: Perform a basename-\/only match if the pattern does not contain any slash characters. That is, {\ttfamily \texorpdfstring{$\ast$}{*}.js} would be treated as equivalent to {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}/\texorpdfstring{$\ast$}{*}.js}, matching all js files in all directories.
\end{DoxyItemize}\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9070}{}\doxysection{\texorpdfstring{References}{References}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_26ab97133b9b78c22feed65a0c527c5dc_autotoc_md9070}
