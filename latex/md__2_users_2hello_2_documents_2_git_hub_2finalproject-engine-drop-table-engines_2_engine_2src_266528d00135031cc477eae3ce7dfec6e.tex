At build time, the N-\/\+API back-\/compat library code will be used only when the targeted node version {\itshape does not} have N-\/\+API built-\/in.

The preprocessor directive {\ttfamily NODE\+\_\+\+ADDON\+\_\+\+API\+\_\+\+DISABLE\+\_\+\+DEPRECATED} can be defined at compile time before including {\ttfamily \doxylink{napi_8h_source}{napi.\+h}} to skip the definition of deprecated APIs.
