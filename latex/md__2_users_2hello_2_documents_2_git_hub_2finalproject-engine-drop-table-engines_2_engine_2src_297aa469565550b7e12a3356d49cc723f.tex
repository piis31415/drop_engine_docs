The object representing the components of the Java\+Script request being made.

The {\ttfamily \doxylink{class_napi_1_1_callback_info}{Napi\+::\+Callback\+Info}} object is usually created and passed by the Node.\+js runtime or node-\/addon-\/api infrastructure.

The {\ttfamily \doxylink{class_napi_1_1_callback_info}{Napi\+::\+Callback\+Info}} object contains the arguments passed by the caller. The number of arguments is returned by the {\ttfamily Length} method. Each individual argument can be accessed using the {\ttfamily operator\mbox{[}\mbox{]}} method.
