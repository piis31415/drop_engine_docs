You know that {\ttfamily mode} property on the {\ttfamily fs.\+Stat} object that you probably usually just ignore? Well there\textquotesingle{}s acutally a lot of information packed into that number.

The specific information includes\+:


\begin{DoxyItemize}
\item What the \href{http://en.wikipedia.org/wiki/Unix_file_types}{\texttt{ "{}file type"{}}} of file it is
\item Whether or not the \href{http://en.wikipedia.org/wiki/Setuid}{\texttt{ {\ttfamily setuid} and {\ttfamily setgid} bits}} are set
\item Whether or not the \href{http://en.wikipedia.org/wiki/Sticky_bit}{\texttt{ {\ttfamily sticky} bit}} is set
\item The \href{http://en.wikipedia.org/wiki/File_system_permissions}{\texttt{ {\itshape read}, {\itshape write}, and {\itshape execute} permissions for the {\itshape owner}, {\itshape group} and {\itshape others}}}
\end{DoxyItemize}

This module helps you extract that information.

All the getters are also setters, which change the {\ttfamily mode} property appropriately. This is useful for when you have to build up your own {\ttfamily fs.\+Stat} object for whatever reason (like when implementing a FUSE filesystem.\hypertarget{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_273662626481080bea025d331f76bf6d9_autotoc_md9910}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__2_users_2hello_2_documents_2_git_hub_2finalproject-engine-drop-table-engines_2_engine_2src_273662626481080bea025d331f76bf6d9_autotoc_md9910}
